🕵️‍♂️ 1. أدوات الاستكشاف والاستطلاع (RECON) 🕵️‍♂️
==================================================

🔍 أدوات فحص الشبكات:
• Nmap - أقوى أداة لفحص الشبكات والـ Ports
• Wireshark - تحليل حزم الشبكة واكتشاف الأنشطة المشبوهة
• Dracnmap - واجهة رسومية فوق Nmap تسهّل تنفيذ المسح
• Red Hawk - أداة PHP شاملة لفحص المواقع والثغرات
• Nikto - فاحص مواقع ويب لاكتشاف ثغرات خطيرة
• OWASP ZAP - واجهة رسومية لفحص تطبيقات الويب
• Burp Suite CE - وكيل اعتراض وفحص لكل تفاصيل تطبيق الويب
• BloodHound - تحليل علاقات Active Directory واكتشاف مسارات الهجوم
• mitmproxy - وكيل اعتراض متقدم لفحص حركة مرور HTTPS
• Gobuster - أداة سريعة للبحث عن المجلدات والملفات المخفية
• ffuf - أداة Fuzzing سريعة للويب مكتوبة بلغة Go
• Nuclei - محرك فحص سريع وقابل للتخصيص للثغرات المعروفة
• Subfinder - أداة سريعة لاكتشاف النطاقات الفرعية من مصادر متعددة
• httpx - أداة سريعة لفحص HTTP probes مع دعم متقدم للـ HTTP
• Naabu - أداة سريعة لفحص المنافذ مكتوبة بلغة Go
• Dirsearch - أداة للبحث عن الأدلة والمجلدات المخفية على المواقع
• Amass - أداة شاملة لمسح البنية التحتية لنطاق معين
• Dnsenum - جمع معلومات DNS شاملة عن هدف معين
• Fierce - أداة تقليدية لكنها فعالة جدًا لفحص DNS
• DNSRecon - أداة فحص تفصيلي لخدمات DNS ونقاط الضعف
• EyeWitness - أداة أخذ لقطات شاشة للويب وتحليل الخدمات
• Aquatone - أداة لجمع صور لصفحات الويب تلقائيًا خلال مرحلة Recon
• PacketFence - نظام إدارة شبكة NAC (Network Access Control) مفتوح المصدر
• EtherApe - أداة مراقبة حركة الشبكة بتصوير رسومي للعقد والاتصالات
• GVM (Greenbone Vulnerability Manager) - منصة إدارة ثغرات مفتوحة المصدر
• OpenVAS - أداة مفتوحة المصدر لمسح الثغرات مبنية على GVM
• WPScan - ماسح ثغرات لمنصات ووردبريس
• Subjack - أداة فحص ثغرات Subdomain Takeover
• MassDNS - أداة مسح DNS عالية الأداء
• dnsx - أداة DNS شاملة لأداء المسح والاستعلامات
• SubOver - أداة فحص Subdomain Takeover
• Assetfinder - أداة لاكتشاف النطاقات الفرعية تلقائيًا باستخدام مصادر عامة
• AltDNS - أداة تولّد permutations لأسماء النطاقات للعثور على subdomains مهملة
• Findomain - أداة فحص subdomains بسرعة عالية جدًا باستخدام Rust
• Knockpy - أداة Python لجمع subdomains واستغلال ثغرات DNS
• DNSDumpster - خدمة استخبارات DNS تجلب سجلات وأنظمة الشبكة العامة
• crt.sh - أداة/موقع لفحص الشهادات الرقمية المرتبطة بنطاق معين
• FOFA - منصة استعلام استخباراتي صيني شبيه بـ Shodan وCensys
• CTFR - أداة فحص subdomains عبر استغلال قاعدة بيانات الشهادات
• Sublert - أداة مراقبة تغييرات DNS عبر الوقت
• LeakLooker - أداة OSINT تبحث عن قواعد بيانات Firebase المكشوفة
• GitHub-Dorks - سكربت يستخدم dorks للبحث عن أسرار مخزنة في GitHub
• Intel Owl - منصة تحليل مركزي تجمع تقارير من عدة مصادر OSINT
• PhoneInfoga - أداة OSINT متخصصة في أرقام الهواتف
• Censys - محرك بحث عن الأجهزة المكشوفة والبنية التحتية للإنترنت
• Amass (Passive Mode) - استخدام Amass في الوضع السلبي لجمع DNS/WHOIS دون كشفك

🔍 أدوات Subdomain Discovery:
• Sublist3r - يبحث ويجمع Subdomains من أكثر من مصدر
• Corsy - أداة لفحص مشاكل CORS (Cross-Origin Resource Sharing) في المواقع
• Gasmask - أداة لإدارة ملفات hosts متعددة في بيئة التطوير

🎯 2. أدوات استغلال الثغرات (EXPLOITATION) 🎯
==================================================

⚔️ أدوات الاختراق الأساسية:
• Metasploit Framework - منصة متكاملة لاستغلال الثغرات وتطوير Payloads
• Aircrack-ng - تحليل وكسر كلمات سر شبكات Wi-Fi
• sqlmap - أداة مؤتمتة لاكتشاف واستغلال ثغرات SQL Injection
• XSStrike - أداة هجومية ذكية ضد XSS بـ Fuzzer متطور
• Hydra - تخمين كلمات السر للبروتوكولات المختلفة SSH, FTP, HTTP...
• John the Ripper - كسر كلمات المرور المشفّرة بأسلوب سريع
• Hashcat - كسر كلمات السر باستخدام GPU بشكل رهيب السرعة
• BeEF - استغلال المتصفحات واستخدامها كـ zombies
• Empire - منصة Powershell & Python لاختراق Windows وPost-Exploitation
• SET (Social-Engineer Toolkit) - إطار لهجمات الهندسة الاجتماعية والتصيّد
• Cobalt Strike - منصة متقدمة للاختراق والـ Post-Exploitation
• Responder - أداة لاستغلال بروتوكولات الشبكة مثل LLMNR وNBT-NS
• Mimikatz - أداة استخراج كلمات السر من الذاكرة
• CrackMapExec - أداة إدارة اختراق الشبكات
• Veil-Framework - أداة توليد Payloads لتجاوز مضادات الفيروسات
• Sn1per - إطار عمل لاختبار الاختراق يحتوي على عدة أدوات موحدة
• Veil-Evasion - أداة توليد Payloads مخفية لتجاوز مضادات الفيروسات
• PentestGPT - أداة تعتمد على LLM لتنفيذ pentest أوتوماتيكي بثبات عالي
• PTF (Penetration Testers Framework) - إطار عمل لتثبيت بيئة أدوات Pentest محترفة
• Crowbar - أداة brute forcing متعددة البروتوكولات (SSH, RDP, VNC...)
• PowerSploit - مجموعة سكريبتات PowerShell للـ post-exploitation والهجوم
• BloodHoundSharp - مكتبة C# لتوليد وتحليل بيانات BloodHound
• SharpHound - أداة جمع بيانات Active Directory بلغة C# جزء من BloodHound
• Impacket - مكتبة Python للتعامل مع بروتوكولات الشبكة (SMB, MSRPC...)
• Evil-WinRM - أداة لفتح جلسات Windows Remote Management (WinRM) في البيئات المستهدفة
• BloodHound.py - نسخة Python لتفاعل مع BloodHound AD data
• CrackStation - أداة كسر كلمات السر تعتمد على قواميس ضخمة جاهزة
• Shellter - أداة حقن شيفرة ديناميكية لملفات PE
• MSFVenom - أداة توليد Payloads ضمن Metasploit Framework

🔧 أدوات توليد القواميس:
• Crunch - مولد قواميس كلمات المرور القابلة للتخصيص
• CeWL - أداة لاستخراج كلمات من مواقع الويب لإنشاء قواميس

• Binary Ninja - منصة تحليل ثنائية متقدمة تجارية مع واجهة رسومية

• dnSpy - محرر ومفكك IL لتطبيقات .NET مع دعم التصحيح والتحرير
• PE-bear - أداة تحليل ملفات PE مع تحليل AI للـ entropy ونقاط الضعف
• ImHex - محرر Hex عرضي مفتوح المصدر متعدد المنصات مع دعم scripting
• Binwalk - أداة لاكتشاف واستخراج الملفات أو البيانات المدمجة داخل الصور
• Immunity Debugger - Debugger لنظام Windows يدعم Python scripting لتحليل الثغرات
• API Monitor - أداة مراقبة واسترجاع استدعاءات API في التطبيقات أثناء التنفيذ
• Process Hacker - أداة مراقبة عمليات النظام في Windows مع قدرات كشف وتحليل متقدمة
• PEiD - أداة كشف Packers وcryptors المستخدمة في ملفات PE
• Scylla - أداة لاستعادة جداول استيراد DLL من ملفات packed
• CFF Explorer - أداة لتحليل ملفات PE، تحتوي على محرر هيكلي شامل
• Hopper - Decompiler لأنظمة macOS وLinux، يسمح بتحليل التعليمات البرمجية الأصلية
• JaDx - Decompiler لتحويل ملفات APK إلى شفرة Java قابلة للتحليل
• Ropper - أداة للبحث عن gadgets في ملفات التنفيذ لتسهيل بناء ROP chains
• Pwndbg - إضافة GDB قوية لتحليل الثغرات مع تحسينات لواجهة المستخدم

🔍 أدوات تحليل الكود:

• STEG‑Detector - أداة لكشف البيانات المخفية في الصور والصوت والفيديو

• Stegano-Thing - أدوات فحص steganography وmetadata متعددة
• steg-tools - أدوات تحليل metadata وsteghide متعددة

• zsteg - يكشف البيانات المخفية في ملفات PNG, BMP باستخدام تحليل LSB
• OpenStego - أداة Java مجانية لإخفاء البيانات داخل الصور مع دعم watermarking

• Stegano (Python) - مكتبة Steganography تخفي البيانات داخل PNG باستخدام LSB
• Snowdrop - أداة Watermarking ونصوص steganography لإخفاء الرسائل داخل نصوص غير مرئية
• Stegoshare - أداة Java لإخفاء بيانات في ملفات WAV مع دعم متعددة
• OutGuess - أداة Steganography عامة تدعم ملفات JPEG وBMP وAU
